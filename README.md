st - simple terminal
---
 st is a simple terminal emulator for X which sucks less.

Keys
---
 This build of st is set to-


Mod Key
---

    Mod4, i.e Super (Default Mod)
    
    Ctrl+Shift (TermMod Key)
   
Actions
---

    Copy:        Ctrl+Shift+c
    Paste:       Ctrl+Shift+v
    
    Color:       Ctrl+Shift+p
    
    Zoom:
      - In:      Ctrl+Shift+pgup
      - Out:     Ctrl+Shift+pgdn
      - Reset:   Ctrl+Shift+home
      
    Scroll:
      - Up:      Alt+Ctrl+k
      - Down:    Alt+Ctrl+j
    
    Url:
      - Select:  Super+l
      - Open:    Super+u
